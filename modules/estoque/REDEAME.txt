######CONTROLE DE ESTOQUE##############
Versão 2.01
Autor: Jonathan Jaci

Correção de alguns erros no script
Adição do sistema de login

1 - Instalando o controle.

- Para instalar o controle basta copiar os arquivos dentro de uma pasta no servidor.
- Execute o arquivo db.sql para criar as tabelas e padrões de dados.
- Acesse o arquivo DB.php na pasta application/controler e configure os dados de conexão com seu BD.
- Acesse o arquivo db.php na pasta principal e configure os dados de usuario, o usuario padrão e admin senha: admin.

2 - Opcões


O sistema de estoque permite que seja cadastrado clientes, formecedores e produtos.
Além disos é permitidoo cadastro de categorias de produtos que é carregado via ajax na hora de dar entrada ou 
saida de um produto.
O sistema também possui um aviso de estoque minimo que avisa sempre que um produto estive em quantidade
abaixo da desejável em estoque.
O sistema de estoque permite que seja cadastrado usuario para acesso ao sistema.



3 - Compatibilidade.


O sistema foi testado no Google Chrome Versão 36 em resolução 1366 x 768.
